# Virtual Bank 

Virtual Bank final project 

## Installation

First of all you need to get all required dependencies for both Front-End and Back-End by opening their respected Integrated Terminals and typing:

```bash
npm install

```
You also need to create *.env* file inside Back-End directory. The example is provided with *.env.sample* ( You can also just change the name of *.env.sample* to *.env* )
## Usage

```bash
Using Postman you can register a new "User" with > "POST" > your local address + "/api/user/register" and by providing:
local address example: "http://localhost:3000/api/user/register"

# Key: userName
# Value: <your desired user name>
"Username must be between 3 and 20 characters"
"Username must be unique"

# Key: password
# Value: <your desired password>
"Password must be at least 6 characters"

```
